# 5. User Stories

## 5.1 Admin User Stories

**US-A1**
As an Admin, I want to manage users in the platform.

**US-A2**
As an Admin, I want to assign a role (Coordinator or Volunteer) so responsibilities are clearly separated.

**US-A3**
As an Admin, I want to deactivate users to prevent unauthorized access.

---

## 5.2 Coordinator User Stories

**US-C1**
As a Coordinator, I want to register to the app.

**US-C2**
As a Coordinator, I want to create events so volunteers can participate.

**US-C3**
As a Coordinator, I want to specify required skills so I can assign suitable volunteers.

**US-C4**
As a Coordinator, I want to see who signed up for an event so I can manage attendance.

---

## 5.3 Volunteer User Stories

**US-C1**
As a Volunteer, I want to register to the app.

**US-V1**
As a Volunteer, I want to see available events so I can choose where to help.

**US-V2**
As a Volunteer, I want to sign up for events so I can participate.

**US-V3**
As a Volunteer, I want to view my participation history so I can track my contributions.
