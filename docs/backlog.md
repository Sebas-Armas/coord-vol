# 4. Feature Backlog (High-Level)

## Phase 1 – Core Platform (MVP)

**Authentication & Users**

* User registration (Admin-created)
* Login with JWT
* Role-based access control

**Admin Features**

* Create users
* Assign role: Coordinator or Volunteer
* Activate / deactivate users

**Coordinator Features**

* Create and edit events
* Define required skills
* View volunteer signups

**Volunteer Features**

* View available events
* Sign up for events
* View participation history

**System**

* EN / ES language switch
* Dockerized services
* CI pipeline with tests

---

## Phase 2 – Operational Enhancements

* Email notifications
* Event capacity limits
* Volunteer availability calendar
* Coordinator dashboard (KPIs)

---

## Phase 3 – Advanced (Optional)

* Audit logs
* Export reports (CSV)
* API rate limiting
* Observability (basic metrics)
