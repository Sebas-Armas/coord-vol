# Product Overview

A web platform that allows organizations to manage volunteers and coordinators, organize events, and track participation.

## Supported Roles (Mutually Exclusive)

- **Admin** – Configures the system and users
- **Coordinator** – Manages events and volunteers
- **Volunteer** – Participates in events

A user **cannot be both Coordinator and Volunteer at the same time.**

## Languages

- English (EN)
- Spanish (ES)

The UI must support **internationalization (i18n)** and allow language switching.
